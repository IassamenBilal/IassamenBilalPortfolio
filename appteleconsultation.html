Application web pour la téléconsultation

Ce projet fait partie des projets que j'ai réalisé en tant que devéloppeur web chez la boite informatique SADEEM informatique.

C'est une application pour la téléconsultation qui permet de rechercher un medecin et prendre un rendez-vous en ligne pour faire une consultation soit au cabinet soit en téléconsultation

Grâce à cette application l'utilisateur s'inscrire en tant que particulier ou bien en tant que medecin

Le particulier peut prendre un rendez-vous chez le medecin pour une consultation.

Le medecin a ses horaires de travail ainsi que les tarifs de consultations il peut les mettre à jour sur son profile

Le particulier sur son profile peut voir la liste de ses rendez-vous à venir ou bien les rendez-vous passés, il peut aussi modifier un rendez-vous
le medecin possède son planning il peut voir les rendez-vous qu'il a dans la journée

Technologies:
  • - HTML
  • - CSS
  • - React JS
  • - Node JS
  • - MySQL
  • - Google Cloud
